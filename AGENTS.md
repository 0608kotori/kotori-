# ASTRONAUTS / 宇航员 Site Rules

## Frozen design baseline

- Read `DESIGN-SPEC.md` before changing site structure, styling, content,
  interaction, or motion.
- `DESIGN-SPEC.md` was explicitly approved and frozen by the user on
  2026-07-19. It is the authoritative source for the site's design direction,
  page hierarchy, component placement, visual tokens, and acceptance criteria.
- Do not add or relocate decorative components, change the page hierarchy, or
  deviate from the frozen visual and motion rules without explicit user
  approval.
- For future React Bits additions, prefer the official TypeScript and CSS
  source supplied by the user. Review and adapt it locally before use; do not
  install a component package, CLI, or new runtime dependency without separate
  explicit approval.
- Do not fabricate people, achievements, outcomes, partners, statistics, or
  contact details. Hide incomplete evidence sections or label their status
  honestly.
- Deployment always requires separate explicit user approval.
